# Changelog

## [1.0] — 2026-07-01

### Added
- GRUB bootloader simulation with boot option selection
- Login prompt with credential authentication (`admin` / `iusearchbtw`)
- Interactive REPL shell (`admin@arch ~ #`)
- Mock `sudo pacman -Syu` system update animation
- `whoami` command displaying the current user
- `clear` command (cross-platform)
- `run-app` command to launch external applications
- Modular project structure with `login/`, `bootloader/`, and `files/` packages
